预测模型介绍

四种模型涵盖经典统计方法到集成学习，各有独立的数学基础。 所有模型共享相同的 22 维特征输入，采用递归多步策略生成 46 天预测。

LightGBM 全局模型 · 全量 SKU

梯度提升决策树。每轮新树拟合前一步预测的损失梯度残差， 逐步纠正误差。按叶优先生长与 GOSS 单侧梯度采样显著提升训练效率。

ŷi = Σk=1100 η · fk(xi)   (η = 0.05)
Gain = ½[GL²/(HL+λ) + GR²/(HR+λ) − (GL+GR)²/(HL+HR+λ)] − γ
w* = −G / (H + λ)
100 棵树 深度 6 31 叶节点 GOSS 采样
随机森林 全局模型 · 全量 SKU

决策树的 Bagging 集成。每棵树在数据的随机自助采样与特征随机子集上独立训练， 通过对不相关预测器取均值大幅降低预测方差。

ŷ = 1⁄B Σb=1B Tb(x)   (B = 50 棵树)
Var(ŷ) = ρσ² + 1−ρ⁄Bσ²
ρ = 树间相关系数；当 ρ → 0 时方差趋近于 0
50 棵树 深度不限 自动特征采样
Prophet 单品模型 · 按需拟合

Meta 开源的加法分解模型，将时序信号拆解为分段线性趋势 （含自动变点检测）、Fourier 级数季节性分量与节假日效应三部分。

y(t) = g(t) + s(t) + h(t) + εt
g(t)：分段线性趋势 + 自动变点
s(t) = Σn[ancos(2πnt/P) + bnsin(2πnt/P)]
P = 7（周季节性周期）
加法模式 周季节性 结果内存缓存
Holt-Winters 单品模型 · 按需拟合

三重指数平滑法，用三个加权方程分别追踪水平、趋势与季节性分量， 平滑参数 α、β、γ 逐 SKU 最优化拟合。

lt = α(yt/st-m) + (1−α)(lt-1+bt-1)  水平
bt = β(lt−lt-1) + (1−β)bt-1  趋势
st = γ(yt/lt) + (1−γ)st-m  季节
ŷt+h = (lt + h·bt) · st−m+h
周期 m = 7 αβγ 最优化 三级容错回退
OLS（岭回归） 未纳入组合

使用全部 24 维特征的 L2 正则化线性回归。因无法捕捉促销与销量之间 的乘法式非线性交互，CV FA 仅 69.5%，较朴素基准低 18 个百分点， 未纳入集成模型。

ŷ = Xβ  ·  β* = argminβ ‖y − Xβ‖² + λ‖β‖²
λ = 1.0（岭正则项）
Ridge α = 1.0 24 个特征 CV FA：69.5%
集成模型 LGB + RF + HW

对 LightGBM、随机森林和 Holt-Winters 三个模型的输出取简单均值， OLS 因表现不佳（低于基准 18 pp）被排除。各模型误差相互独立时， 集成方差以 1/K 比例收缩，预测更稳定。

ŷens = 1⁄3(ŷLGB + ŷRF + ŷHW)
Var(ŷens) ≈ 1⁄K · σ²
（各模型误差独立时成立）
简单均值 K = 3 方差 ∝ 1/K

LLM 自动特征工程

待开发

当前 22 维特征集由领域知识人工设计。下一阶段将引入 大语言模型（LLM） 作为自动特征工程智能体：给定预测任务的自然语言描述与数据字段定义， LLM 自动提出、评估并筛选最具预测力的特征组合，无需人工逐一指定。

① 提案

LLM 读取数据字典与任务描述，自动生成候选特征清单——包括滞后窗口、 交叉项、外部信号等——并按预期重要性排序输出。

② 评估

对每个候选特征自动构造并计算交叉验证折的 SHAP 重要性或置换重要性， 量化其对预测精度的实际贡献。

③ 决策

LLM 综合评估结果，输出最终特征集，并以自然语言给出每个特征 入选或剔除的业务解释，实现可解释的自动化特征筛选。

📐 POC 准确率计算方法 — FA 公式

FA  =  1 − |F − A| / ((F + A) × 2)

F = 该 SKU 当周预测销量合计  ·  A = 该 SKU 当周实际销量合计

① 若 F 或 A < 0，则取 0
② 若 F = A = 0，则 FA = 100%
③ FA 取值范围限定在 [0%, 100%]

加权汇总步骤（共 3 步）：

1. 将逐日预测与实际销量聚合为周合计， 按周期划分（7月1–7日、8–14日、15–21日、22–28日……）
2. 各 SKU 权重 = 该周开始前 4 周的实际销量合计
3. 加权平均 FA → 品类层级（8802–8807）→ 全局汇总
加权 FA = Σ(FAi × wi) / Σwi

wi = SKUi 过去 4 周实际销量
